Crystal Collector Game

INSTRUCTIONS:

~ At the start of a game a random number will be generated in the "Random Number Box."
~ Each one of the four gems at the bottom are worth a different number between 1-12. When a gem is clicked that amount will be added to the "Current Score Box."
~ The objective of the game is to match your score with the random generated number, if matched you win.
~ If your score goes over the random number you will lose the game.
~ When a new game begins there will be a new random number given and the four gems will have new values assigned to them.
